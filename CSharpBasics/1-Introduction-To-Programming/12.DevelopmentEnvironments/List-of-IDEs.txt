Visual Studio: Visual Studio is an integrated development environment (IDE) from Microsoft. It is used to develop computer programs for Microsoft Windows, web applications and web services.
Built-in languages include C, C++ and C++/CLI (via Visual C++), VB.NET (via Visual Basic .NET), C# (via Visual C#), and F# (as of Visual Studio 2010).

Eclipse: Eclipse is written mostly in Java and it is used to develop Java aplications. It has open sourse.

NetBeans: NetBeans is IDE for developing mostly in Java but also in PHP, C, C++ and HTML.

JDeveloper: Jdeveloper is a freeware IDE supplied by Oracle Corporation. It offers features for development in Java, XML, SQL and PL/SQL, HTML, JavaScript, BPEL and PHP.
